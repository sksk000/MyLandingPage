### 概要
以下のページを模倣し、言語・ライブラリ学習の為に作成いたしました。
https://vanillawebprojects.com/projects/modal-menu-slider/

### 実行方法
#### Github上で確認する場合
#### リポジトリを取得して確認する場合
1. git cloneコマンドを使用し、https://github.com/sksk000/MyLandingPage.git を指定して任意のフォルダにリポジトリを取得します。
2. 任意のフォルダ\MyLandingPage\html内にあるmain.htmlを開くと実行することができます。
3. デモのような画面が表示されることが確認できます。

### 使用方法
- 左側にあるハンバーガーメニューをクリックすると目次が表示されるようになります。
